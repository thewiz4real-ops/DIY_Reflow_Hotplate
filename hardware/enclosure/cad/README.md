# Enclosure CAD

The current enclosure source files are grouped under `brain-box/`.

Formats currently included:

- Fusion 360 archive export: `reflow-plate-brain-box.f3d`
- 3MF export: `reflow-plate-brain-box.3mf`
- OBJ mesh export: `reflow-plate-brain-box.obj`
- OBJ material file: `reflow-plate-brain-box.mtl`

No STL or STEP export is checked in yet.
