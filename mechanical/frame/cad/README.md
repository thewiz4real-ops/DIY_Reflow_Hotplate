# Frame CAD

The current main plate CAD files are grouped under `reflow-plate/`.

Formats currently included:

- Fusion 360 archive export: `reflow-plate.f3d`
- 3MF export: `reflow-plate.3mf`
- OBJ mesh export: `reflow-plate.obj`
- OBJ material file: `reflow-plate.mtl`

No STEP or STL export is checked in yet.
