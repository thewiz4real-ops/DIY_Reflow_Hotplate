# Frame

Main structure made from:
- 50 x 50 cm aluminum base plate
- T-slot profiles
- drilled and tapped mounting holes

Suggested things to document:
- exact profile lengths
- exact hole spacing
- wheel setup
- microscope mount dimensions
- control box bracket dimensions

- `reflow-plate.f3d`
- `reflow-plate.3mf`
